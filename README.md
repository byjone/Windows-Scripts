# 🪟 Windows Scripts

## 💻 About this repository

This repository contains a collection of **Windows automation and administration scripts** developed as part of my **Systems Administration (ASIR)** studies.

It focuses on improving efficiency in Windows environments by automating common tasks, supporting system maintenance, and practicing real-world administrative scenarios.

The main goal is to build practical skills in Windows system management and scripting.

---

## 🧠 What this repository is about

This repository focuses on practical Windows administration tasks such as:

- System maintenance and optimization tasks
- User and local group management
- File system operations and automation
- Network configuration and diagnostics
- Software and process management
- System monitoring and troubleshooting
- Basic security and configuration tasks

Each script is designed to solve common tasks in Windows environments.

---

## 🧱 Technologies used

- 🪟 Windows 10 / Windows 11
- ⚡ PowerShell scripting
- 🧰 Command Prompt (CMD)
- 🔧 Windows built-in administration tools
- 📊 Event Viewer and system logs
- 🌐 Networking utilities (ipconfig, ping, etc.)
